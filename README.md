# BuggyJS 🏎️

> Un gioco di corse retro fatto in **un singolo file HTML**.  
> Perché usare un engine da 200MB quando bastano 844 righe di JavaScript?

---

## Come si gioca

| Tasto | Azione |
|-------|--------|
| ↑ / W | Accelera (e prega) |
| ↓ / S | Frena (troppo tardi, probabilmente) |
| ← → / A D | Sterza |
| 1 – 4 | Scegli difficoltà (1 = passeggiata, 4 = funerale) |

---

## Feature

- Grafica **pixelata** d'altri tempi — volutamente, giuro
- Audio sintetico via Web Audio API — suona come un motorino con l'influenza
- Highscore salvato in `localStorage` — sopravvive al refresh, non alla vergogna
- Zero dipendenze, zero build step, zero scuse

---

## Come si avvia

```bash
# Metodo 1: trascina buggyboy.html nel browser
# Metodo 2: fai doppio click
# Metodo 3: apri il terminale, scrivi un server HTTP, senti subito il rimorso
```

---

## Requisiti

- Un browser (anche quello che usi solo per scaricare Chrome)
- Dita funzionanti
- Spirito competitivo almeno pari a quello di un Tamagotchi
